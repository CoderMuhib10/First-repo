how to commit a repository

  1. create a new repository in github
    2. [note: give the repo a meaningful and similar to the project folder name]
  2. Open CMD or Vs code terminal(press ctrl+jto open)

    3. If you are doing this for the first time-

  3. open cmd in the directory which you want to push.
    4. to open-
    • first run the following command: cd\ folder name or you can just type cmd in the folder url to open that directory in cmd.
      • tip:after typing some letters of that folder name you can press tab to aoutomatically detect the folder name.
  4. run the commands serially showing in the github page.
    • git init
    • git add README.md or git add .
    • git commit -m "write a meaningful commit message"
      • note: as you are doing for the first tim they will ask you for your info. you've to run the command that they will provide you and give your name and email adress the 2 commands they'll give you-
      • git config --global user.email "
      • git config --global user.name "
      if they're not showing prompt after running those 2 commands press w+q
    • Now again run the same command
      • tip: you can go to your previous commands by pressing the down arrow key.
    • git branch -M main
    • git remote add origin https://github.com/CoderMuhib10/First-repo.git
    • git push -u origin main
      • note: after running this command you'll get a pop wanting your aouthentication.simply aouthenticate that by clicking
    Here you go! you're all done!!

    What's now??

    Now everytime you start a new project follow these following steps

    1. create a new repository and a local folder in your computer
    2. open terminal in your vs code by pressing ctrl+j
    3. now run these commands in the terminal
      • git init
      • git add README.md or git add .
      • git commit -m "write a meaningful commit message"
      • git branch -M main
      • git remote add origin https://github.com/CoderMuhib10/First-repo.git
      • git push -u origin main
    4. now whenever you edit the file/folder you only have to run these 3 commands to commit the changes-
      • git add .
      • git commit -m "write a meaningful commit message"
      • git push
